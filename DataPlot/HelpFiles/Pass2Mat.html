Work in Progress

 

 

 

Input File: The .txt file created with pass3200 software by exporting the data

Output Directory: Choose an empty directory for this with a relevant identifying name. This is where the roughly 600 data files will end up, so in order to avoid confusion don’t mix this data with other stuff.

Output File ID: Choose something that will identify the files, typically the flight number or tail number of the aircraft. Whatever you type in here will form part of the output filenames

 

The right hand panel is for adding new parameters or making your own smaller set of parameters.

In general just use the LOAD button and load the “weaponsbus.cfg” config file which will extract all the parameters ion the weapons bus.

 

Approximate Start: If you want to skip the first part of the input file, just guess the amount of minutes you want to skip from start of recording and enter it in here. If you want to stop the processing before all the data has been parsed, just press CANCEL, when you think you have enough data.

The “seconds” edit box in the bottom left corner will show how many seconds have been processed, so you can use this together with your Approximate start time to extract only a chunk of data out of the middle of the file.

The blue process bar at the bottom will show approximate process until processing the file is done
