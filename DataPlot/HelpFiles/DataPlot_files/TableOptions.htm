Changing a Parameter                                                       To Top

 

Select the parameter cell in the Parameter column to be changed, so that it is highlighted and right click with the mouse. This will bring up a list box as below. Select the Edit option.

                                                                

 

 

This will bring up the SELECT FILE box.

You can’t actually edit a parameter, you have to select a different parameter from the available list of files given in the  SELECT FILE box

 

 

Deleting a Parameter                                                          To Top

Select the parameter cell in the Parameter column to be deleted and right click with the mouse
