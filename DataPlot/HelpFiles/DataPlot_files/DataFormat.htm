Data Format

Plot1553 is Matlab based and can only use data in the following formats:

  1. A matlab data file ( .mat ) consisting of 2 columns where the first column contains the Time stamps in milli seconds and the second column contains the actual decimal data. It is accepted that the data would already be converted into the correct format according to the specified ICD document. One such tool to prepare the data into the correct format is Pass2Mat
  2. A text file ( .txt ) consisting of 2 columns where the first column contains the Time stamps in milli seconds and the second column contains the actual decimal data. These 2 columns must just be seperated by a white space.
  3. It can also be put into a matlab structure ( .struct ) although this becomes a problem when the data is too much. This format is the same as for the .mat files except that it is not in individual matlab files but all in one file under a structure “Filename.Data.Parameter”

 

The Type of input file  (one of the 3 above) is selected in Plot1553 under the Input Type panel inside the Project panel by clicking on the appropriate radio button.

 

Note:

In general Try to Only use the .mat files  as the .txt files takes up too much space and the .mat struct falls over when the flight data is too big.

File Format

The files are typically stored in a specific directory using the following structure:

 

DIR\PARAM.FID.EXT

Where :

DIR        is the path to the directory

PARAM   is usually the parameter name as defined in the Aircraft ICD

FID        is usually the 3 digit flight number pertaining to the data

EXT        can be .mat or  .txt

 

Example:

 

c:\Rooivalk Data\Aircraft_673\flight_090\mss_az.090.mat

 

DIR        c:\Rooivalk Data\Aircraft_673\flight_090\

PARAM   mss_az

FID        090

EXT        mat
