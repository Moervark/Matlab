Graphics Properties

To change the Figure properties in terms of Font and marker Size click on the Tools dropdown menu and select Graph Properties.

It will bring up the following Screen:

 

 

Title Font Size

Changes the font size of the Title on the Figure

 

Axes Font Size

Changes the font size of the axis indexing as well as the legend font.

It does not change the font size of the axis Labels though.

 

Marker Size

Changes the Size of the Symbols, so this only has an effect if you plot the graphs using Symbols from the Symbol drop down menu in the Table
