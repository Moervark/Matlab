Complex Bit Plot

Continuing from the example above we will now write the Bitplot slightly different to give a more user friendly output

First we tell the Bitplot that we want to plot from Bit 0 to Bit 2 by typing in the Bit column of the Table “0-2”

The tool will now automatically convert these 3 bits to its decimal value and assign enough space for it on the Figure. We can now also add a better description in the description column. I have added another Bit 0  - Fire in Magenta  just to show the complex bit plotting more clearly.

 

 

 

This result in a plot as seen below,  the single Bits (Fire and Plotting a single Bit)  is as before but the TCU Mode plot can now have 7 distinct values.

 

However in this representation it is still difficult to guess exactly at which one of the 7 values the plot is at any given time.

 

So lets ask Bitplot to add automatic grid lines, by typing the following in the Bit column: 0-2-0. This tells Bitplot that we want to plot the decimal value of Bits 0-2 and the -0 tells it to add automatic gridlines. With the -0 option it will take the maximum decimal value possible for the amount of bits and print a grid line at each integer, In this example it means 3 bits => 23-1 = 7 => lines at [ 0 1 2 3 4 5 6 7 ].

 

 

 

The result:

 

Now you can easily read the decimal value at any given point.

Note:

Unfortunately you can’t change the labels of the different levels – Something for a next release

 

In this case it is not very relevant but when there is too many grid lines cluttering the plot you can specify to Bitplot that for instance you want to only see 3 gridlines apart from the 0 grid line at [ 0  2.33  4.66  7 ]. Do this by changing the automatic gridline option -0 to a specific number, in this case -3 because you want 3 gridlines.
